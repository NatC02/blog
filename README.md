## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/NatC02/blog
   ```
2. Install npm packages
   ```sh
   npm install
   ```
3. Start developing 
   ```sh
   gatsby develop
   ```
4. Build the public folder for deployment 
   ```sh
   gatsby build
   ```

## Deploy:


1. Deploy to Github Pages
   ```sh
   npm run deploy
   ```
